# EduSpark - Ignite Your Learning

EduSpark is a premium, modern bookstore application built with Next.js and MongoDB. It features a stunning UI, dynamic book loading, and a complete checkout flow for eBooks.

## 🚀 Live Demo
**Website**: [https://eduspark-bookstore.vercel.app](https://eduspark-bookstore.vercel.app)

## ✨ Features
- **Modern UI/UX**: Crafted with vanilla CSS and Framer Motion for smooth, premium animations.
- **Dynamic Catalog**: Full integration with MongoDB Atlas for real-time book listings.
- **Book Details**: Detailed pages for every book including summaries, metadata, and related suggestions.
- **Smart Cart**: Persistent shopping cart with GST calculations and tax breakdown.
- **Checkout Pipeline**: Complete digital checkout flow with UPI QR code integration.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Technology Stack
- **Frontend**: Next.js 15+, React 19, Tailwind CSS, Lucide React
- **Backend**: MongoDB (Mongoose ORM), Next.js API Routes
- **State Management**: Zustand
- **Animations**: Framer Motion
- **Deployment**: Vercel

## 📦 Getting Started

### Prerequisites
- Node.js 20+
- MongoDB Atlas account

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ayushpandey29/EduSpark.git
   cd EduSpark
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Set up environment variables:
   Create a `.env.local` file and add your MongoDB URI:
   ```env
   MONGODB_URI=your_mongodb_connection_string
   ```
4. Run the development server:
   ```bash
   pnpm dev
   ```
